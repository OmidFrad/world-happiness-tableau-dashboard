# World Happiness Tableau Dashboard

## Project Overview

This project analyzes the World Happiness Report dataset from 2015 to 2023 using Tableau Public.

The goal of this project is to explore happiness scores across different countries and understand how key indicators such as GDP per capita, social support, freedom, generosity, and healthy life expectancy relate to national happiness levels.

## Dataset

The project uses World Happiness Report data from 2015 to 2023.

Main fields used in the analysis:

- Country
- Region
- Year
- Happiness Score
- GDP per Capita
- Social Support
- Freedom to Make Life Choices
- Generosity
- Healthy Life Expectancy

## What I Built

I created several worksheets, dashboards, and a Tableau Story.

### 1. Happiness Status Overview

This dashboard compares the top 10 happiest countries and the bottom 10 least happy countries based on average happiness score from 2015 to 2023.

### 2. GDP per Capita and Happiness

This dashboard analyzes the relationship between GDP per capita and happiness score for the top 10 happiest countries.

### 3. Factors Affecting the Most Happy Countries

This dashboard explores how freedom, social support, generosity, and healthy life expectancy relate to happiness among the happiest countries.

### 4. Factors Affecting the Least Happy Countries

This dashboard applies the same analysis to the least happy countries to compare patterns and differences.

### 5. Tableau Story

A Tableau Story was created to present the main findings in a structured narrative format.

## Methodology

1. Imported the World Happiness dataset into Tableau Public.
2. Prepared and checked the required fields.
3. Created groups for the top 10 happiest and bottom 10 least happy countries.
4. Built highlight tables, maps, line charts, and dual-axis combination charts.
5. Combined worksheets into interactive dashboards.
6. Created a Tableau Story to summarize the analysis.
7. Published the final workbook on Tableau Public.

## Key Insights

- Happier countries usually show higher GDP per capita and stronger social support.
- Freedom and healthy life expectancy appear to be important factors related to happiness.
- Least happy countries generally perform weaker across multiple well-being indicators, not only GDP per capita.
